** How to create a virtual environment:
>> at first TurnOff your Real Time Protection from Windows Security
>> Type commands:
  1. python -m venv myenv ------  { python -m venv (your_environment_name) }
  2. myenv\Scripts\activate  { To activate your Virtual Environment }
  3. deactivate  { To deactivate V.E. }
